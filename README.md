# PaizoSessionParser
A Python script do parse information of all Organized Play Sessions of Paizo systems you have ever participated in

**EDIT THE SOURCE TO SET YOUR OUTPUT LOCATION IN LINE 9**

The tool uses three freely available Python modules: requests, BeautifulSoup and lxml.

To install those requirements, run
```
pip install -r requirements.txt
```

To run the script:

```
python PaizoSessionParser.py
```
